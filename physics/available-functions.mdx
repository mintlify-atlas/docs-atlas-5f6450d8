## Overview

Particle Simulator injects the entire JavaScript `Math` library into your formulas, giving you access to a comprehensive set of mathematical functions and constants.

<Note>
All functions from JavaScript's `Math` object are automatically available - no prefixes or imports needed!
</Note>

## Variables

| Variable | Type | Description |
|----------|------|-------------|
| `t` | number | Elapsed time in seconds since simulation start |
| `x` | number | Current X position of the particle |
| `y` | number | Current Y position of the particle |
| `z` | number | Current Z position of the particle |

## Trigonometric Functions

Standard trigonometric functions (angles in radians):

<CodeGroup>
```javascript Sine and Cosine
// Sine wave
sin(t)           // Range: [-1, 1]

// Cosine wave
cos(t)           // Range: [-1, 1]

// Circular motion
10 * cos(t)      // X component
10 * sin(t)      // Y component

// Phase-shifted oscillation
sin(t * 0.5)     // Slower frequency
sin(t + PI/4)    // 45° phase shift
```

```javascript Tangent
// Tangent (unbounded)
tan(t)           // Range: (-∞, ∞)

// Periodic pulses
tan(sin(t))      // Bounded by sin

// Angle wrapping
atan2(y, x)      // Range: [-π, π]
                 // Returns angle from origin
```
</CodeGroup>

### Available Functions:

- **`sin(x)`** - Sine of x (x in radians)
- **`cos(x)`** - Cosine of x
- **`tan(x)`** - Tangent of x
- **`asin(x)`** - Arcsine (inverse sine)
- **`acos(x)`** - Arccosine (inverse cosine)
- **`atan(x)`** - Arctangent (inverse tangent)
- **`atan2(y, x)`** - Two-argument arctangent (returns angle in correct quadrant)

## Hyperbolic Functions

Hyperbolic trigonometric functions (useful for damping and bounded growth):

<CodeGroup>
```javascript Tanh (Bounded)
// Hyperbolic tangent
tanh(x)                    // Range: (-1, 1)
                          // Smooth bounded function

// Soft clipping
20 * tanh(sin(t))         // Bounded oscillation

// Sigmoid-like behavior
tanh(t - 5)               // Smooth step at t=5
```

```javascript Sinh and Cosh
// Hyperbolic sine (exponential growth)
sinh(x)                   // Range: (-∞, ∞)

// Hyperbolic cosine
cosh(x)                   // Range: [1, ∞)

// Catenary (hanging chain)
cosh(x) - 1
```
</CodeGroup>

### Available Functions:

- **`sinh(x)`** - Hyperbolic sine: `(e^x - e^(-x)) / 2`
- **`cosh(x)`** - Hyperbolic cosine: `(e^x + e^(-x)) / 2`
- **`tanh(x)`** - Hyperbolic tangent: `sinh(x) / cosh(x)`
- **`asinh(x)`** - Inverse hyperbolic sine
- **`acosh(x)`** - Inverse hyperbolic cosine
- **`atanh(x)`** - Inverse hyperbolic tangent

### Why Use Hyperbolic Functions?

<Tabs>
  <Tab title="Smooth Saturation">
    **`tanh(x)`** provides smooth bounded output:
    
    ```javascript
    // Instead of hard clipping:
    min(max(value, -1), 1)  // Sharp corners
    
    // Use tanh for smooth saturation:
    tanh(value)             // Smooth curve
    ```
    
    Perfect for:
    - Limiting forces without discontinuities
    - Soft transitions
    - Sigmoid activation
  </Tab>
  
  <Tab title="Damping">
    **Hyperbolic functions** model physical damping:
    
    ```javascript
    // Overdamped motion
    -z * 0.5 + 20 * tanh(sin(t))
    ```
    
    The `tanh` term provides:
    - Bounded external forcing
    - Prevents instability
    - Natural-looking damping
  </Tab>
  
  <Tab title="Growth & Decay">
    **`sinh/cosh`** model exponential processes:
    
    ```javascript
    // Growth that stabilizes
    10 * tanh(t / 5)
    
    // Catenary curve
    cosh(x / 10) - 1
    ```
  </Tab>
</Tabs>

## Exponential and Logarithmic

Exponential growth/decay and logarithmic functions:

<CodeGroup>
```javascript Exponential
// Natural exponential
exp(x)                    // e^x

// Exponential decay
exp(-t * 0.5)            // Decay with half-life

// Gaussian-like
exp(-x^2 - y^2)          // Radial decay from origin
```

```javascript Logarithms
// Natural log
log(t + 1)               // Avoid log(0)

// Base-10 log
log10(t + 1)

// Logarithmic spiral
log(sqrt(x^2 + y^2) + 1)
```

```javascript Power and Root
// Square root
sqrt(x^2 + y^2)          // Distance from origin

// Cube root
sign(x) * abs(x)^(1/3)   // Handles negative values

// Arbitrary power
x^2.5                     // or x**2.5
```
</CodeGroup>

### Available Functions:

- **`exp(x)`** - Euler's number raised to x: `e^x`
- **`log(x)`** - Natural logarithm (base e)
- **`log10(x)`** - Base-10 logarithm
- **`log2(x)`** - Base-2 logarithm
- **`sqrt(x)`** - Square root: `√x`
- **`cbrt(x)`** - Cube root: `∛x`
- **`pow(x, y)`** - x raised to power y (can also use `x^y` or `x**y`)

## Utility Functions

Practical functions for common operations:

<CodeGroup>
```javascript Absolute Value & Sign
// Absolute value
abs(x)                   // |x|

// Always positive force
abs(cos(t*0.2))         // Pulsing without negative

// Sign function
sign(x)                  // -1, 0, or 1
sign(x) * abs(x)^2      // Preserve sign in power
```

```javascript Rounding
// Floor (round down)
floor(t)                 // Integer part

// Ceiling (round up)
ceil(t)

// Round to nearest
round(t * 10) / 10      // Round to 1 decimal

// Truncate
trunc(x)                 // Remove decimal part
```

```javascript Min/Max
// Minimum
min(x, y)               // Smaller value
min(abs(x), 10)         // Clamp to max 10

// Maximum
max(x, y)               // Larger value
max(z, 0)               // Clamp to min 0

// Clamping
min(max(x, -5), 5)      // Range: [-5, 5]
```
</CodeGroup>

### Available Functions:

- **`abs(x)`** - Absolute value: `|x|`
- **`sign(x)`** - Sign of x: -1, 0, or 1
- **`floor(x)`** - Round down to nearest integer
- **`ceil(x)`** - Round up to nearest integer
- **`round(x)`** - Round to nearest integer
- **`trunc(x)`** - Remove fractional part
- **`min(a, b, ...)`** - Smallest value
- **`max(a, b, ...)`** - Largest value
- **`random()`** - Random value in [0, 1)

### Practical Examples:

<CardGroup cols={2}>
  <Card title="Pulsing Force" icon="heart-pulse">
    ```javascript
    -x * abs(cos(t*0.2))
    ```
    Spring force that pulses between 0 and full strength
  </Card>
  
  <Card title="One-Way Force" icon="arrow-right">
    ```javascript
    max(0, -z)
    ```
    Only applies when z is positive (acts like a floor)
  </Card>
  
  <Card title="Stepwise Force" icon="stairs">
    ```javascript
    floor(t) * 5
    ```
    Changes in discrete steps
  </Card>
  
  <Card title="Random Noise" icon="shuffle">
    ```javascript
    (random() - 0.5) * 2
    ```
    Random force in [-1, 1]
  </Card>
</CardGroup>

## Advanced Techniques

### Composition

Combine functions for complex behaviors:

```javascript
// Modulated oscillation
sin(t) * cos(t * 0.1)

// Exponentially decaying oscillation
exp(-t * 0.2) * sin(t * 3)

// Logarithmic spiral
log(t + 1) * cos(t)
log(t + 1) * sin(t)

// Bounded chaotic-like
tanh(sin(t) * cos(t * 1.7) * 3)
```

### Avoiding Singularities

<Warning>
Be careful with division by zero and invalid inputs!
</Warning>

```javascript
// BAD: Can divide by zero
x / sqrt(x^2 + y^2)

// GOOD: Add epsilon
x / sqrt(x^2 + y^2 + 0.001)

// BAD: log(0) is undefined
log(t)

// GOOD: Offset to avoid zero
log(t + 1)

// BAD: Negative square root
sqrt(x)

// GOOD: Use absolute value
sqrt(abs(x))
```

### Performance Optimization

<Tip>
The formula cache system means complex formulas are only compiled once. Don't be afraid to use sophisticated expressions!
</Tip>

```javascript
// This is fine - compiled once:
-x / (sqrt(x^2 + y^2 + z^2 + 0.1)^3) + 
 sin(t * 0.5) * tanh(sqrt(x^2 + y^2) / 10)
```
