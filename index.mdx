## What is Particle Simulator?

Particle Simulator is a real-time physics simulation and visualization engine that lets you define particle motion through custom mathematical formulas. Built with React and Three.js, it provides an interactive 3D environment for exploring complex force equations and particle dynamics.

Whether you're studying physics, prototyping simulations, or just exploring beautiful mathematical patterns, Particle Simulator makes it easy to visualize particles in motion.

## Quick Example

Define a particle affected by a Lissajous resonance pattern:

```typescript
// Position
x₀ = 10, y₀ = 0, z₀ = 0

// Forces
Fx = -x
Fy = -2.25 * y
Fz = sin(t * 0.5) * 5
```

Watch as the particle creates beautiful 3D patterns in real-time.
